## 06. State

> Terraform은 인프라 및 구성에 대한 상태를 저장해야 합니다.
> 이 상태는 Terraform에서 실제 리소스를 구성에 매핑하고 메타 데이터를 추적하며 대규모 인프라의 성능을 향상 시키는데 사용 됩니다.
> 이 상태는 기본적으로 terraform.tfstate 라는 로컬 파일에 저장되지만 원격으로 저장할 수도 있으므로 팀 환경에서 더 잘 작동합니다.
